## How to Run

1. Clone the repo

2. Open in Visual Studio

3. Press F5 to run
 
## Project Structure

- **Models/Board.cs** - Game grid and logic

- **Players/Player.cs** - Abstract base class

- **Players/HumanPlayer.cs** - Human input handling

- **Players/ComputerPlayer.cs** - AI logic

- **Controllers/GameController.cs** - Game flow

- **Views/ConsoleView.cs** - Display and UI
 
## OOP Concepts Used

- Abstraction: Player abstract class

- Encapsulation: Private fields in Board

- Inheritance: HumanPlayer and ComputerPlayer extend Player

- Polymorphism: GetMove() works differently per player type
